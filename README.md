## Репозитарий содержит некоторые проекты, выполненные в процессе обучения профессии "Специалист по Data Science" в Яндекс.Практикуме

## Описание проектов


| Название проекта      | Описание              | Библиотеки            |
| :-------------------- | :-------------------- | :-------------------- |
| [Прогнозирование количества заказов такси на следующий час в аэропортах](https://github.com/YuliyaKotova/projects/tree/main/predict_taxi_orders_timeseries) | Исследование временного ряда. Дифференцирование ряда. Построение регрессионных моделей для прогнозирования количества заказов такси. Выбор лучшей модели, анализ остатков лучшей модели. | pandas, matplotlib, numpy, sklearn, lightgbm, statsmodels, scipy |
| [Анализ спроса пассажиров авиакомпании](https://github.com/YuliyaKotova/projects/blob/main/avia_sales_market_research) | Исследование спроса пассажиров на рейсы по разным направлениям в обычное время и во время проведения фестивалей. Проверка статистической гипотезы: в фестивальные недели спрос пассажиров на билеты не меняется. | pandas, matplotlib, numpy, scipy |
| [Прогнозирование оттока клиентов в банке](https://github.com/YuliyaKotova/projects/tree/main/predict_bank_clients_exit) | Решение задачи классификации: уйдёт клиент из банка в ближайшее время или нет. Исследовательский анализ данных, в том числе исследование баланса классов целевого признака. Построение моделей без учета дисбаланса классов, улучшение качества моделей с учетом дисбаланса классов разными методами. Выбор лучшего метода и лучшей модели.  | pandas, matplotlib, seaborn, numpy, sklearn, time |
